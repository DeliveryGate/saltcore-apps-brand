## 4. Full Description (ready to paste)
Product bundles are one of the most effective ways to increase average order value — but setting them up on Shopify usually requires complex workarounds or expensive third-party apps with confusing pricing.

BundleBox keeps it simple. Build bundles from your existing products, set the discount, and BundleBox handles the rest. Discounts are applied at checkout automatically using a native Shopify Function — meaning faster checkout, no code, and no discount code management.

**Three bundle types to suit every merchant:**

**Fixed Bundle** — A curated set of products sold together at a discount. "The Starter Pack" with 3 specific products for 15% off. Customers see the bundle on the product page and add it with one click.

**Mix-and-Match Bundle** — Let customers build their own bundle from a selection. "Choose any 4 sauces for 20% off." Perfect for food businesses, beauty brands, gift retailers.

**Volume Bundle** — Automatic tiered discounts for buying more of the same product. Buy 3, get 10% off. Buy 6, get 20% off. Applied automatically, no codes required.

**How it works:**
1. Create a bundle in the BundleBox admin — name it, pick the products, set the discount
2. The bundle widget appears automatically on product pages (via theme extension)
3. Customers add the bundle to cart — the discount applies instantly via Shopify Functions
4. No discount codes, no checkout confusion, no margin leakage from shared codes

**What merchants love about BundleBox:**
- Works with existing products — no duplicate variants or shadow products
- Bundle widget matches your theme's style automatically
- Discounts apply before the customer reaches checkout — no surprises
- Manage all bundles from one clean admin dashboard
- Bundle overview dashboard — track active bundles by type

**Built for food merchants, gifting stores, and anyone who wants to sell more per order.**

Perfect for food businesses — create meal deal bundles, sharing platters, family packs, and catering packages. Bundle Builder works for any Shopify store.

### Key Benefits

**1. Increase Average Order Value Without Complicated Offers**
Pre-built and mix-and-match bundles are proven to lift basket size. BundleBox makes them easy to create — no variant duplication, no discount code management, just bundles that sell.

**2. Native Shopify Functions — Fast, Reliable Discounts**
BundleBox uses a Shopify Function to apply discounts, not discount codes or cart scripts. That means discounts appear instantly at checkout, work with all payment methods, and never expire or get shared online.

**3. Three Bundle Types for Every Use Case**
Fixed bundles for curated products, mix-and-match for customer choice, volume tiers for repeat buyers. Most apps offer one approach — BundleBox gives you all three in a single dashboard.
