## 4. Full Description (ready to paste)
### Give every customer exactly the meal they want — without SKU chaos.

Restaurants, delis, meal prep services, and caterers all face the same problem: customers want to customise their order, but Shopify's product variants don't handle multi-category meal configuration. Menu Builder solves this with a fully configurable meal builder that works entirely on your storefront — no custom coding required.

**How it works:**
1. Create a meal builder in the Menu Builder admin (name it, set sizes, add item categories)
2. Add your menu items to each category with prices, allergens, and images
3. Install the Menu Builder theme block on any Shopify page
4. Customers select size, pick items from each category, and see a live allergen summary updating as they build

**What customers experience:**
- Size selector (e.g. Small / Large) with different slot configurations per size
- Category browsing: Base, Main, Veg, Sides, Sauce — you define the categories
- Per-item allergen data displayed inline as options are selected
- Live allergen summary that updates in real time as the customer builds their meal
- Price uplifts shown clearly (e.g. Salmon + £2.00)
- Running total price displayed throughout the build
- Notes field for special instructions and dietary requirements
- Quantity input
- Add to Cart button — passes full meal configuration as Shopify line item properties

**What you configure in the admin:**
- Multiple independent builders (one for lunch, one for dinner, one for event catering)
- Builder name, intro text, and ordering rules
- Size options with custom labels and slot counts per category
- Menu items: name, description, base price, price uplift, allergens, sort order
- Date picker for advance ordering (catering and pre-orders)
- Active/inactive toggle per builder

**Cart integration:**
Customers build their perfect meal and see live allergen and pricing information. When they click Add to Cart, the full meal configuration is passed to Shopify as line item properties — so your order management system receives the complete selection detail.

Cart integration requires a product variant ID to be set in the block settings. You create a generic "Custom Meal" product in Shopify, copy its variant ID, and paste it into the Menu Builder block settings in your theme editor. This takes about two minutes during setup and works with Dawn and all major Shopify themes.

*Note: Cart integration requires merchant configuration of a product variant ID in the block settings. A setup guide is available in the app's admin dashboard.*

**Works with any Shopify theme:**
Menu Builder installs as a native Shopify theme block. Add it to any page using the theme editor — no liquid editing required. One builder per page, or multiple builders on the same page using different block IDs.

## 3 Key Benefits

**1. Offer configurable meals without managing hundreds of variants**
A freedom tray with 3 bases, 5 mains, and 4 sides would require 60+ product variants in standard Shopify. Menu Builder handles this with zero variants — customers configure at checkout, and the selection is passed as line item properties.

**2. Show allergen information live as customers build their meal**
As a customer selects each item, their allergen exposure updates in real time. They can see — before ordering — exactly what allergens are in their configured meal. This is the gold standard for allergen transparency and protects your business from complaints.

**3. Handle catering and advance orders with date picking**
Menu Builder includes an optional date picker for advance ordering. Set how far in advance customers must order, and let customers select their meal delivery or collection date during the build — not at checkout.
